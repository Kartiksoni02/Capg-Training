HTML : used to create and structure web pages.
HyperText : Text with links
Markup : Tags used to structure content

> Invented by Tim Berners-Lee in 1990.

# Search Engine Optimization :
SEO is the process of improving a website to rank higher on search engines and get more traffic.

> A Single Page Application (SPA) loads only one HTML page and dynamically updates content without reloading the page.
SEO is difficult in this. 
At first , it is slow , but after reloading it is faster.
First SPA in 2009.
Example: Gmail, Facebook

How it works:
First request → full page loads
After that → only data (JSON/API) loads
Page does NOT refresh

> A Multiple Page Application (MPA) loads a new HTML page for every request. Easier SEO
Example: Amazon, Flipkart

How it works:
Each click → sends request to server
Server returns a new HTML page
Whole page reloads

Attributes : give extra information about a tag.

1. Global Attributes : Can be used with any tag
id      Unique identifier
class   to Group elements
style   Inline CSS
title   Tooltip text , when we move cursor on title

2. Specific Attributes : Used with specific tags
href → <a>
src → <img>
alt → <img>

3. Boolean Attributes : No value needed (true/false)
disabled
checked
readonly

4. Form Attributes :
action	    Where data is sent
method	    How data is sent (GET / POST)
target	    Where to open result
